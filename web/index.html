API Reference

🌐 Base URL
https://your-domain.railway.app

AUTHENTICATION

Include your API key in every request header:

Authorization: Bearer YOUR_API_KEY

Generate your key from the API page above.

RESPONSE FORMAT

{
  "success": true,
  "data": { ... },
  "message": "OK"
}

ENDPOINTS

GET /api/v1/balance

Get user token and gem balances.

{ "success": true, "balance": { "tokens": 5000, "gems": 10 } }
GET /api/v1/user/stats

Get detailed user stats and API status.

{ "success": true, "balance": 5000, "tokens": 5000, "gems": 10, "apiStatus": "allow" }
POST /api/v1/email/temp/create

Generate a temporary email address. (Costs 5 TC)

{ "success": true, "email": "api_abc123@temp-verify.org", "cost": 5 }
POST /api/v1/email/premium/buy

Buy a Premium Email (gmail or hotmail).

Body: { "type": "gmail" }
{ "success": true, "type": "gmail", "email": "user@gmail.com", "password": "pwd", "cost": 50 }
POST /api/v1/number/request

Request a virtual number for SMS verification.

Body: { "service": "telegram", "country": "usa" }
{ "success": true, "number": "+12025551234", "requestId": "api_req_...", "cost": 15 }
POST /api/v1/store/buy

Purchase an item from the digital store.

Body: { "itemId": "chatgptplus" }
{ "success": true, "itemName": "ChatGPT Plus", "data": "Purchase successful...", "cost": 100 }

ERROR CODES

400 Bad Request / Insufficient balance
401 Invalid or missing API key
403 API access restricted or banned
404 Resource not found / Out of stock
500 Server error — retry later

QUICK EXAMPLE

// JavaScript fetch example
const res = await fetch(
  'https://your-domain.railway.app/api/v1/balance',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
const data = await res.json();
console.log(data.balance.tokens); // 5000

• Free tier: 60 requests/min
• Each API call costs tokens from your balance
• Check balance before bulk operations
